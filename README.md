# PositionIQ

PositionIQ is a sports analytics platform built with Python and Streamlit.

## Current Features

- Odds Converter
    - American
    - Decimal
    - Fractional
    - Implied Probability

- Two-Way No-Vig Calculator
- Market Overround Analysis
- Fair Odds Estimation
- Educational Insights

## Built With

- Python
- Streamlit
- Git
- VS Code
